好些年沒有寫部落格啦，感覺竟然有些生疏。

## 從 WordPress 到 Jekyll

大概五年前剛上大學那會兒有用過 WordPress 來搭建個人網站，WordPress 給我最大的感覺自然還是太臃腫，尤其是當時我還用的是暢銷榜第一名的 avada 主題，雖說可定制性相當不錯，但對於個人網站來說我還是更傾向於極簡風格的頁面，比如我一直都很喜歡的 [zen habits](https://zenhabits.net/) 這種風格，如此一來讀者更能將注意力放在內容上，而非被不必要的動效和版塊干擾到。

至於搭建這個技術部落嘛，本來是打算用 Django 框架從頭開始搭建一個完全符合自己使用需求的 blog 系統，但後來考慮到自己還是個菜鳥程序員（主要原因）還有後期的運行和維護可能會影響自己寫文的心態（次要原因），所以最後還是決定用 Jekyll + GitHub Pages 的方式來快速搭建部落格。至於具體的搭建過程和使用體驗的話，還是之後再給大家分享吧。

## 這個部落格會記錄些什麼？

程序員的部落格當然還是以技術為主啦。雖然自己從開始學習編程到現在也不過是半年多時間，還有太多太多東西是完全不懂（比如我也是最近這一個月時間才開始學習使用 Git 和 Linux 什麼的），但還是想從現在開始記錄自己不斷成長的過程，希望自己可以一直堅持下去吧（笑）。
